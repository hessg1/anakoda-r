# anakoda-r
anakoda-r are R scripts for data analysis in the anakoda project, that allow to connect directly to MIDATA for retrieving and analysing the headache data in the FHIR format.

anakoda-r is part of a bachelor thesis at the Institute for Medical Informatics I4MI at Bern University for Applied Sciences. 
It is created on the base of some scripts from the migren-DV LC2 project.
